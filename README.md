# Информационно-справочная система «Аэропорт»

Мобильное приложение для платформы Android, предназначенное для автоматизации
информационно-справочных операций в аэропорту: поиск рейсов, онлайн-бронирование
билетов и администрирование данных.

## Стек технологий

- **Android-клиент:** Java 17, MVVM, Android Jetpack (ViewModel, LiveData, Navigation)
- **Бэкенд:** Spring Boot 3, Spring Data JPA, Spring Security + JWT
- **База данных:** PostgreSQL 15
- **Взаимодействие:** REST API (Retrofit 2 + Gson)

  # Регистрация и вход

## Регистрация нового пользователя

1. Откройте приложение. На стартовом экране нажмите **«Зарегистрироваться»**.
2. Заполните форму регистрации:
   - **Имя и фамилия** — ФИО в формате «Иванов Иван Иванович»;
   - **Паспортные данные** — серия и номер паспорта;
   - **Телефон** — контактный номер в формате +7XXXXXXXXXX;
   - **E-mail** — адрес электронной почты (используется как логин);
   - **Пароль** — минимум 6 символов;
   - **Подтверждение пароля** — повторите пароль.
3. Нажмите **«Создать аккаунт»**.
4. После успешной регистрации вы будете автоматически перенаправлены
   на главный экран приложения.

> **Важно:** E-mail должен быть уникальным. Если указанный адрес уже
> зарегистрирован в системе, вы получите соответствующее сообщение об ошибке.

## Вход в систему

1. На стартовом экране введите **E-mail** и **Пароль**.
2. Нажмите **«Войти»**.
3. При успешной аутентификации откроется главный экран приложения.

## Выход из аккаунта

Перейдите в **«Личный кабинет»** → нажмите **«Выйти»** в нижней части экрана.
Сессия будет завершена, приложение вернётся на стартовый экран.

## Возможные ошибки

| Сообщение | Причина | Решение |
|-----------|---------|---------|
| «Неверный e-mail или пароль» | Введены некорректные данные | Проверьте правильность ввода |
| «Пользователь уже существует» | E-mail занят | Используйте другой адрес или войдите в существующий аккаунт |
| «Нет подключения к серверу» | Отсутствует интернет или сервер недоступен | Проверьте подключение к интернету |

# Поиск рейсов

## Описание

Раздел позволяет найти доступные рейсы по заданным критериям.
Список рейсов отсортирован по времени вылета.

## Как выполнить поиск

1. Откройте приложение и перейдите на вкладку **«Рейсы»**.
2. Заполните одно или несколько полей фильтра:
   - **Откуда** — город отправления (например, «Москва»);
   - **Куда** — город прибытия (например, «Санкт-Петербург»);
   - **Дата** — выберите дату из календаря.
3. Нажмите кнопку **«Найти»**.
4. Просмотрите список найденных рейсов.
   Каждая карточка содержит:
   - номер рейса и авиакомпанию;
   - город отправления и прибытия;
   - время вылета и прилёта;
   - цену билета;
   - количество свободных мест;
   - текущий статус рейса.
5. Нажмите на карточку рейса для просмотра подробной информации.

> **Подсказка:** Можно оставить все поля пустыми и нажать «Найти» —
> в этом случае отобразятся все доступные рейсы.

## Статусы рейсов

| Статус    | Описание                        |
|-----------|---------------------------------|
| SCHEDULED | Рейс запланирован по расписанию |
| BOARDING  | Идёт посадка на рейс            |
| DEPARTED  | Рейс вылетел                    |
| ARRIVED   | Рейс прибыл в пункт назначения  |
| DELAYED   | Рейс задержан                   |
| CANCELLED | Рейс отменён                    |

## Детальная информация о рейсе

При нажатии на карточку рейса открывается экран с полной информацией:
- номер рейса, авиакомпания;
- маршрут: город и аэропорт отправления → прибытия;
- дата и время вылета / прилёта;
- терминал и выход на посадку (gate);
- тип воздушного судна;
- цена билета;
- количество доступных мест;
- статус и, при наличии, причина задержки.

Из экрана детальной информации можно перейти к бронированию,
нажав кнопку **«Забронировать»**.

# Бронирование билета

## Требования

Для оформления бронирования необходимо:
- быть зарегистрированным пользователем и войти в систему;
- на выбранном рейсе должно быть не менее одного свободного места;
- рейс должен иметь статус **SCHEDULED** или **BOARDING**.

## Пошаговая инструкция

1. Найдите нужный рейс через раздел [Поиск рейсов](./Поиск-рейсов).
2. Откройте карточку рейса и нажмите **«Забронировать»**.
3. Проверьте автоматически заполненные данные из вашего профиля:
   - ФИО;
   - номер паспорта;
   - контактный телефон.
4. При необходимости скорректируйте данные непосредственно в форме.
5. Ознакомьтесь с итоговой стоимостью бронирования.
6. Нажмите **«Подтвердить бронирование»**.
7. После успешного бронирования система:
   - создаёт запись о бронировании со статусом **CONFIRMED**;
   - уменьшает количество доступных мест на рейсе на 1;
   - присваивает вам номер места;
   - перенаправляет вас в раздел **«Мои бронирования»**.

## Данные бронирования

После оформления вам доступна следующая информация:
- уникальный номер (ID) бронирования;
- номер рейса, маршрут, дата и время вылета;
- присвоенный номер места;
- стоимость;
- статус бронирования;
- дата и время оформления.

## Статусы бронирования

| Статус    | Описание                                  |
|-----------|-------------------------------------------|
| CONFIRMED | Бронирование подтверждено                 |
| CANCELLED | Бронирование отменено пользователем       |
| COMPLETED | Рейс завершён, бронирование закрыто       |

## Ограничения

- Один пользователь не может оформить два бронирования на один и тот же рейс.
- Бронирование на отменённый (CANCELLED) или уже вылетевший (DEPARTED/ARRIVED)
  рейс недоступно.

  # Мои бронирования

## Просмотр списка бронирований

1. Перейдите на вкладку **«Мои бронирования»** в нижнем меню приложения.
2. Отображается список всех ваших бронирований, отсортированных
   по дате вылета (ближайшие — первыми).
3. Каждая карточка содержит:
   - номер рейса и маршрут;
   - дату и время вылета;
   - номер места;
   - стоимость;
   - текущий статус бронирования.

## Просмотр деталей бронирования

Нажмите на карточку бронирования для просмотра полной информации:
- все данные рейса (терминал, gate, тип самолёта);
- ваши данные пассажира, указанные при бронировании;
- статус оплаты и регистрации на рейс.

## Отмена бронирования

1. Откройте детали нужного бронирования.
2. Нажмите кнопку **«Отменить бронирование»**.
3. Подтвердите действие в появившемся диалоговом окне.
4. После отмены:
   - статус бронирования меняется на **CANCELLED**;
   - количество доступных мест на рейсе увеличивается на 1;
   - бронирование остаётся в списке с отметкой «Отменено».

> **Внимание:** Отменить бронирование можно только при статусе **CONFIRMED**.
> Бронирования со статусом **COMPLETED** отмене не подлежат.

# Личный кабинет

## Просмотр профиля

Перейдите на вкладку **«Профиль»** в нижнем меню приложения.
Отображается следующая информация:
- ФИО;
- адрес электронной почты (логин);
- номер паспорта;
- контактный телефон;
- роль в системе (Пассажир / Администратор).

## Редактирование профиля

1. На экране профиля нажмите **«Редактировать»**.
2. Измените необходимые поля:
   - ФИО;
   - номер паспорта;
   - контактный телефон;
   - адрес электронной почты.
3. Нажмите **«Сохранить»**.
4. Изменения применяются немедленно.

> **Примечание:** Изменение пароля в текущей версии приложения
> недоступно через интерфейс. Обратитесь к администратору.

## Выход из аккаунта

На экране профиля нажмите **«Выйти»** в нижней части экрана.
Локальные данные сессии будут очищены, приложение вернётся
на стартовый экран входа.

# Руководство администратора

> Данный раздел предназначен для сотрудников аэропорта с ролью **ADMIN**.
> Функции администратора недоступны обычным пользователям-пассажирам.

## Управление рейсами

### Просмотр всех рейсов

Перейдите в раздел **«Рейсы»**. Администратор видит все рейсы в системе,
включая отменённые и завершённые. Для каждого рейса доступны кнопки
редактирования, обновления статуса и удаления.

### Добавление нового рейса

1. В разделе **«Рейсы»** нажмите **«+ Добавить рейс»**.
2. Заполните обязательные поля:
   - **Номер рейса** — в формате «SU-1234»;
   - **Авиакомпания** — название перевозчика;
   - **Город отправления** и **город прибытия**;
   - **Дата и время вылета** / **прилёта**;
   - **Терминал** и **выход на посадку (gate)**;
   - **Тип воздушного судна** — модель самолёта;
   - **Общее количество мест** — вместимость воздушного судна;
   - **Цена билета** — стоимость в рублях.
3. Нажмите **«Сохранить»**.
4. Рейс появится в расписании с количеством доступных мест,
   равным общему числу мест, и статусом **SCHEDULED**.

### Редактирование рейса

1. Найдите рейс в списке, нажмите **«Редактировать»**.
2. Измените необходимые поля.
3. Нажмите **«Сохранить»**.

> **Примечание:** Изменение количества мест пересчитывает доступные места
> с учётом уже оформленных бронирований.

### Обновление статуса рейса

1. Найдите рейс в списке и откройте его карточку.
2. Нажмите **«Изменить статус»**.
3. Выберите новый статус из списка:
   - SCHEDULED → BOARDING → DEPARTED → ARRIVED
   - Любой статус → DELAYED (укажите причину задержки)
   - Любой статус → CANCELLED
4. При выборе статуса **DELAYED** заполните поле **«Причина задержки»**.
5. Нажмите **«Подтвердить»**.

### Удаление рейса

> ⚠️ **Внимание!** При удалении рейса все связанные бронирования
> также будут безвозвратно удалены. Действие необратимо.

1. Откройте карточку рейса.
2. Нажмите **«Удалить рейс»**.
3. Подтвердите действие в диалоговом окне.

### Просмотр бронирований по рейсу



1. Откройте карточку рейса.
2. Нажмите **«Бронирования»**.
3. Отображается список всех бронирований на данный рейс:
   - ФИО и контакты пассажира;
   - номер места;
   - стоимость;
   - статус бронирования;
   - флаги оплаты и регистрации (чек-ин).

### Просмотр списка пользователей

Перейдите в раздел **«Пользователи»** в административном меню.
Отображается список всех зарегистрированных пользователей системы.

4. Нажмите **«Сохранить»**.

2. Нажмите **«Удалить пользователя»**.
3. Подтвердите действие в диалоговом окне.


# Часто задаваемые вопросы

## Бронирование

**Как отменить бронирование?**
Перейдите в **«Мои бронирования»**, откройте нужное бронирование
и нажмите **«Отменить бронирование»**. Подробнее см.
раздел [Мои бронирования](./Мои-бронирования).

**Можно ли забронировать несколько мест на один рейс?**
Нет, один пользователь может оформить только одно бронирование на рейс.

**Бронирование оформлено, но мест на рейсе не осталось — как такое возможно?**
Это невозможно: система резервирует место в момент подтверждения бронирования
и сразу уменьшает счётчик доступных мест.

**Могу ли я изменить данные пассажира после бронирования?**
Нет. Данные пассажира фиксируются в момент бронирования. Для исправления
обратитесь к администратору.

---

## Рейсы

**Что означает статус DELAYED?**
Рейс задержан. Новое время вылета уточняйте у представителей авиакомпании
или в информационном табло аэропорта. В карточке рейса отображается
причина задержки, если она была указана администратором.

**Можно ли забронировать билет на отменённый рейс?**
Нет. Бронирование на рейсы со статусом CANCELLED или ARRIVED недоступно.

**Почему в поиске не отображаются некоторые рейсы?**
Возможные причины: рейс уже вылетел или завершён (статус DEPARTED/ARRIVED),
либо рейс отменён (CANCELLED). Такие рейсы не выводятся в результатах поиска.

---

## Аккаунт

**Забыл пароль. Что делать?**
В текущей версии приложения функция восстановления пароля через e-mail
не реализована. Обратитесь к администратору системы.

**Как сменить e-mail?**
Перейдите в **«Личный кабинет»** → **«Редактировать»** и измените
поле **E-mail**. Новый адрес начнёт использоваться при следующем входе.

**Нет подключения к серверу — что делать?**
Убедитесь, что устройство подключено к интернету. Если интернет работает,
возможно, сервер временно недоступен — попробуйте повторить позже.
